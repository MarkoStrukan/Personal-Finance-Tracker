# Personal Finance Tracker

A simple command-line tool to track your income and expenses. Add transactions, view summaries, and plot your finances over time.

## Features

- Add transactions with date, amount, category (Income/Expense), and description.
- View transactions within a date range.
- See total income, expenses, and savings.
- Optional line plot of income and expenses over time.
